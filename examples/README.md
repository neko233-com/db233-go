# db233-go 示例项目

这个示例项目展示了如何使用 db233-go 的所有主要功能。

## 功能演示

- ✅ 配置管理
- ✅ 日志系统
- ✅ 数据库连接
- ✅ 实体管理
- ✅ CRUD 操作
- ✅ 事务管理
- ✅ 健康检查
- ✅ 数据迁移

## 运行示例

### 1. 准备数据库

首先创建示例数据库：

```sql
CREATE DATABASE db233_demo;
```

### 2. 运行示例

```bash
cd examples
go run main.go
```

### 3. 查看输出

示例程序会按顺序演示所有功能，并输出相应的日志信息。

## 配置文件

示例使用 `config.yml` 配置文件。你可以根据需要修改数据库连接信息。

## 迁移文件

示例包含一个简单的用户表迁移：

- `001_create_users_table.up.sql` - 创建用户表
- `001_create_users_table.down.sql` - 删除用户表

## 注意事项

- 确保 MySQL 服务正在运行
- 修改配置文件中的数据库连接信息以匹配你的环境
- 示例会创建和删除测试数据，请谨慎运行

## 学习路径

1. 查看 `main.go` 了解基本用法
2. 查看配置文件了解配置选项
3. 查看迁移文件了解数据库迁移
4. 参考主项目的 README 了解更多高级功能
